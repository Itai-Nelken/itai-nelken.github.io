To add this repo to your system, run:

wget -qO- https://itai-nelken.github.io/my-repo/add-repo.sh | sudo bash

Alternatively add this line to /etc/apt/sources.list:

deb [trusted=yes] https://itai-nelken.github.io/my-repo/ /

Remember to run sudo apt update after doing that.
